# Code of Conduct

## Our Pledge

We pledge to make participation in this project a welcoming and harassment-free experience for everyone, regardless of background, identity, experience level, or personal characteristics.

## Our Standards

Examples of behaviour that contributes to a positive community include:

- Showing empathy, patience, and respect for differing viewpoints and experience levels.
- Giving actionable technical feedback focused on code and ideas rather than people.
- Accepting responsibility, apologising when appropriate, and learning from mistakes.
- Prioritising the health of the project and its users over individual preferences.

Unacceptable behaviour includes harassment, intimidation, discrimination, trolling, personal attacks, sexualised language or imagery, publishing private information, and any conduct that would be inappropriate in a professional setting.

## Scope

This code applies in repository issues, pull requests, discussions, reviews, and other project spaces. It also applies when someone officially represents the project in public.

## Enforcement

Report unacceptable behaviour privately to the repository owner through the contact methods on the [maintainer's GitHub profile](https://github.com/georgepwall1991). Do not open a public issue containing sensitive or identifying information.

Project maintainers will review reports promptly and fairly, protect the reporter's privacy where possible, and may edit or remove contributions or temporarily or permanently restrict participation when conduct is inappropriate, threatening, or harmful.

If a report concerns the repository owner, contact [GitHub Support](https://support.github.com/) or use GitHub's abuse-reporting tools.
